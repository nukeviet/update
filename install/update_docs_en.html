• - If you use a non-default theme, have not updated to 4.5.00, then update according to the instructions Instructions to upgrade the theme from NukeViet 4.4.02 to NukeViet 4.5.00. If you use the default theme or have updated to 4.5.00 before, skip this step.
  • - If you are using a non-default theme, have not updated to 4.5.01, update according to the instructions Instructions to upgrade the compatible interface from NukeViet 4.5.00 to NukeViet 4.5.01. Note: This step is necessary for your theme that uses block global.QR_code.php, if you don't use it or don't have it, you don't need to do this step..
  • - Additional externally installed modules need updating to work on NukeViet 4.5.01, (If you have already upgraded those modules to NukeViet 4.5.00, skip this step) see update instructions here
  • - Go to Configuration => Security Settings to set up Cross-Site and Domain Restriction functions according to your needs.
  • - Go to Configuration => Manage Modules and click on the News module name and all its virtual modules so that the system recognizes the new function. If there are multiple languages, the languages need to be processed in turn. For example: admin/index.php?language=vi&nv=modules&op=show&mod=news
  • - Go to Configuration => Manage Modules and click on the name of the Users module and all its virtual modules so that the system can recognize the new function. If there are multiple languages, the languages need to be processed in turn. For example admin/index.php?language=vi&nv=modules&op=show&mod=users
  • - If the site uses external libraries such as `phpoffice/phpspreadsheet` through composer, you need to composer update
  • - These can be done after the upgrade. You should save the instructions above for later use.
