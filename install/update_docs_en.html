Read the note in this document to upgrade external modules to version 4.4.05 (optional)

If your website was previously upgraded to a version lower than 4.4.03 then you need to see the issues below. If before upgrading your website from 4.4.03 or higher, you do not need to care about the issues below.

  • - Additional installed modules need upgrade to work on NukeViet 4.4.05, see update instructions here
  • - If you use a non-default theme please upgrade to 4.4.05 follow the instruction here
  • - Modules used for version 4.4.05 if not upgrade it will still run, but will be poor in terms of security and no canonical link tag
  • - The interface if not upgraded to it will still run but without some new features. If you use block global.QR_code.php, you need to upgrade according to the upgrade instructions interface above. If you don't upgrade that block won't display.
  • - These things can be done after upgrading. You should save the two instructions above for later use
